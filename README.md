Este repositório contém os resultados gerados no meu trabalho de conclusão de curso.

Nele encontram-se:

A função que gera o processo de ramificação no arquivo funcao_ramificacao.R.
É necessário executá-la antes de qualquer outro arquivo.

As bibliotecas do RStudio utilizadas no arquivo bibliotecas.R. 
É necessário instalar e carregá-las para obter os resultados do trabalho.

Os arquivos que geram as imagens do trabalho. Eles estão divididos por pasta referente à distribuição utilizada.
Dentro de cada pasta há 3 arquivos.

Exemplo utilizando a pasta 'Binomial':
parametros_binomial.R: neste arquivo geram-se as variáveis que armazenam as replicações do processo de ramificação, as informações de média e variância para C com distribuição Binomial.
graficos_binomial.R: neste arquivo geram-se os gráficos de média e variância para C com distribuição Binomial.
pext_binomial.R: neste arquivo estão as contas para encontrar a probabilidade de extinção e os gráficos para C com distribuição Binomial.

As pastas 'Geométrica' e 'Poisson' seguem o mesmo padrão dos arquivos da pasta Binomial, para C com as respectivas distribuições.
